# Chromosomal-Introgression-Visualization
This repository contains the R code file that goes along with the YouTube tutorial video to demonstrate how to create rough chromosomal introgression plots in R. The input files required should be in ABH format, which can be obtained from VCF files using TASSEL 5 (also demonstrated in the YouTube video. Check it out!).
